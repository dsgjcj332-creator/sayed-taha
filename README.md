# El-Sayed Taha | Interactive Profile

This is an interactive profile/portfolio application built with React, Vite, Tailwind CSS, and Firebase.

## Features
- **Interactive UI:** Smooth animations using Framer Motion.
- **Dynamic Content:** Real-time data fetching using Firebase Firestore.
- **Responsive Design:** Mobile-first approach with Tailwind CSS.
- **Data Visualization:** Radar charts using Recharts.
- **Social Integration:** Direct links to WhatsApp, LinkedIn, Email, and custom apps.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd <your-repo-directory>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Setup Firebase:
   - Create a Firebase project.
   - Add your Firebase configuration to a `firebase-applet-config.json` file in the root directory.

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```
